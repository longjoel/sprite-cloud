Games Vault Architecture

Signaling-only web · WebRTC P2P · OAuth · gv-server on home PC

gv-web (Cloud)

  • • Next.js 15 + React 19
  • • OAuth (GitHub, Google)
  • • WebSocket signaling only
  • • No video relay — lightweight
  • • PostgreSQL: users, games, sessions

gv-server (Home PC)

  • • Rust binary, persistent WS to gv-web
  • • Listens for play signals
  • • Spawns gv-worker per game
  • • Manages ROM library

gv-worker (Home PC)

  • • Rust binary (subprocess)
  • • Libretro core + ROM → video
  • • WebRTC P2P to browser
  • • STUN/TURN for NAT traversal
