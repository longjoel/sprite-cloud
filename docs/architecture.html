Games Vault Architecture

gv-web as relay · gv-server on home PC · gv-worker per game

gv-web (Cloud)

  • • Next.js 15 + React 19
  • • Serves website + gv-player
  • • Signaling: relays play requests
  • • TURN + WebRTC video relay

gv-server (Home PC)

  • • Rust binary
  • • Persistent WS to gv-web
  • • Manages ROM library
  • • Spawns gv-worker per game

gv-worker (Home PC)

  • • Rust binary (subprocess)
  • • Loads libretro core
  • • Reads ROM from library
  • • WebRTC video to gv-web
